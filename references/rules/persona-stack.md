# Persona Stack (from `suno.md`)

A persona should be a **character dossier**, not a label. Vague personas produce inconsistent vocals.

## The 4 layers

1) **Demographics + timbre**  
Age, gender, voice type, core character (bright/dark/airy/raspy/etc.).

2) **Technical delivery**  
Enunciation, phrasing, breath control, vocal techniques, vibrato, grit, rap vs sung, etc.

3) **Emotional context**  
Detached/passionate/vulnerable/aggressive/numb/etc.

4) **Sonic anchor (artist comparison)**  
Reference points that give Suno a clear target (1–2 comparisons max).

## Template

Write as a single compact line:

```text
<age/gender/voice type + timbre>, <technical delivery>, <emotional context>, reminiscent of <anchor 1> with <anchor 2>-like atmosphere.
```

## Example (from `suno.md`)

```text
Female contralto, androgynous, cold, monotone delivery, sharp enunciation, emotionally numb, sinister tone, reminiscent of Grimes with HEALTH-like atmosphere.
```
