# Cover-based remastering template (from `suno.md`)

Use when the user already has a song and wants higher quality. `suno.md` recommends **Cover** over built-in Remaster for quality improvement.

## Minimal Cover prompt

Keep it minimal: genre + high fidelity recording + professional mastering. Avoid extra style descriptors unless you want drift.

```text
Genre: <original genre> with high fidelity recording and professional mastering.
```

Optional (if needed):

```text
Instrumentation: Acoustic drums with realistic sound.
Mastering: Clean, modern, professional sound.
```

## Suggested parameters

- Weirdness: 0
- Style Influence: 100
- Audio Influence: 100
