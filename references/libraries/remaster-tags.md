# Remaster tag quicklist (from `suno.md`)

Use: add a few tags to **Displayed Lyrics** before hitting Remaster.

## Effective remaster tags

- `[high_fidelity]`
- `[studio_mix]`
- `[analog_warmth]`
- `[crystal_clarity]`
- `[punchy_dynamics]`
- `[tape_saturation]`
- `[vocal_depth]`
- `[smooth_transients]`

## Example stacks

- Vintage warmth: `[tape_saturation]` + `[analog_warmth]` + `[smooth_transients]`
- Crisp modern clarity: `[crystal_clarity]` + `[transient_detail]` + `[punchy_dynamics]`
